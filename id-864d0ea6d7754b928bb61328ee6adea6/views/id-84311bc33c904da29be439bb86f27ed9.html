Belastningsreduksjon (historisk) ()
Selve jobben med å redusere belastningen for næringslivet foregår i etatene: FORENKLING: Elektronisk innrapportering Uttrekk av data fra bedriftenes administrative systemer GJENBRUK AV DATA: Etatene skal ikke spørre etter informasjon som de kan hente fra andre kilder – eller informasjon de selv allerede har. Skjemaet forenkles fordi noen data kan hentes fra andre kilder Etaten fyller ut skjemaet på forhånd med kjente data SAMORDNING: Rapporteringstidspunktet endres og koordineres med lignende rapporteringer Etater samarbeider om felles skjema SANERING: Oppgaveplikten opphører fordi lov eller forskrift blir endret Oppgaveplikten opphører fordi informasjon kan hentes fra andre kilder Noen av initiativene krever lov- eller forskriftsendring.
